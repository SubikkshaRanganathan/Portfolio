QuizDB

QuizDB is an interactive and database-driven quiz application developed using MySQL and Java. It's designed to provide an engaging and educational experience for users, making learning fun and effective. With a vast collection of questions and a user-friendly interface, QuizDB facilitates learning through quizzes, helping students and enthusiasts acquire knowledge in an enjoyable manner. As the developer, my role involved designing the database, creating the Java application, and ensuring a seamless user experience.

YouTube Transcript Summarizer

The YouTube Transcript Summarizer is an AI-powered tool built in Python and using the Streamlit framework. It serves as a valuable resource for content consumers and creators. This application takes video transcripts and condenses them into concise summaries, making video content more accessible and time-efficient. As the developer, my responsibilities included implementing the natural language processing algorithms, creating the user interface, and ensuring the tool's accuracy and utility.

Task Management System

The Task Management System is a comprehensive platform developed with Golang, React.js, Tailwind CSS, and MySQL. It is designed to streamline task management, enhance productivity, and promote collaboration within teams and organizations. The system offers features such as task assignment, progress tracking, and reporting. As a key developer on this project, my contributions included designing the backend in Golang, implementing the frontend using React.js and Tailwind CSS, and ensuring seamless database integration with MySQL.

Educational Training Module

The Educational Training Module is an immersive learning experience that focuses on organic chemistry models. It is powered by Three.js and integrated with AWS to provide a cutting-edge educational tool. This module offers an interactive 3D exploration of complex organic chemistry structures, enhancing the understanding and retention of critical concepts. My role in this project included creating 3D models, optimizing the performance with Three.js, and deploying the module on AWS, ensuring a seamless educational experience.
